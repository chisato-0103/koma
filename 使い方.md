# 使い方ガイド

## 起動方法

```bash
npm run dev
```

---

## 初回起動時：初期設定

アプリを初めて起動すると、2ステップのセットアップ画面が表示されます。

### ステップ 1：各コマの時間を設定

| コマ | デフォルト |
| --- | --- |
| 1限 | 08:50 〜 10:30 |
| 2限 | 10:40 〜 12:20 |
| 昼休み | 12:20 〜 13:10 |
| 3限 | 13:10 〜 14:50 |
| 4限 | 15:00 〜 16:40 |
| 5限 | 16:50 〜 18:30 |
| 6限 | 18:40 〜 20:20 |

- 各行の時刻をクリックして変更できます
- **2限と3限の間が自動的に昼休みとして表示されます**
- 設定はあとから ⚙ ボタンでいつでも変更できます

「次へ →」を押して次のステップへ。

### ステップ 2：最初の学期を作成

- 学期名を入力します（例：`2026年前期`）
- 「完了」を押すとメイン画面に移動します

## 授業の追加

1. 時間割の**空きセル（+が表示されているマス）をクリック**
1. 以下の情報を入力します

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| 授業名 | ✅ | 例：線形代数 |
| 教室 | — | 例：A棟101 |
| コマ数 | — | 1コマ（デフォルト）または 2コマ（連続2限使用） |
| Moodle URL | — | MoodleのコースページURL |
| シラバス URL | — | シラバスのURL |
| Teams URL | — | Microsoft TeamsのチームまたはチャンネルURL |

1. 「保存」を押して登録

### 2コマ授業について

- コマ数「2コマ」を選ぶと、そのコマと次のコマをまとめて1つの授業として管理できます
- 時間割グリッドでは2行分を占有するセルとして表示されます
- 2限（昼休みまたぎ）と6限（次限なし）では選択できません
- 次のコマにすでに別の授業が登録されている場合も選択できません

## 授業の編集・削除

1. 登録済みのセル（授業名が表示されているマス）をクリック
1. 内容を編集して「保存」
1. 削除する場合は「削除」ボタン（赤）を押す

### URL をブラウザで開く

Moodle・シラバス・Teams のいずれかURL を入力すると、入力欄の横に「開く」ボタンが表示されます。クリックするとデフォルトブラウザで開きます。

### ノートを記録する

授業モーダルの「📝 ノートを開く」ボタンをクリックすると、専用のノートページに移動します。

#### 回数ごとのノート管理

ノートページ上部の**セッションバー**で、授業の回数（第1回・第2回…）を切り替えられます。

- 「**＋**」ボタンをクリックすると新しい回を追加できます
- セッションタブの右側の「**×**」をクリックすると、その回のノートを削除できます
- 初回開封時は「第1回」が自動的に作成されます
- 各回のノートは個別の `.md` ファイルとして保存されます

#### 表示モード

右上のトグルボタンで3つのモードを切り替えられます。

| モード | 説明 |
| --- | --- |
| 編集 | テキストエリアで Markdown を編集 |
| 分割 | 左半分にエディタ、右半分にリアルタイムプレビューを同時表示 |
| プレビュー | 編集内容を Markdown でレンダリングして確認 |

#### Tab キーインデント

- `Tab` キーを押すと2スペースのインデントを挿入できます
- `Shift + Tab` で先頭の2スペースを削除できます

#### 自動保存

- 1.5秒間入力が止まると自動保存されます（ステータスバーに表示）
- セッションを切り替える前にも自動的に保存されます

#### 対応 Markdown 記法

| 記法 | 説明 |
| --- | --- |
| `# 見出し` | h1〜h6 の見出し |
| `**太字**` | 太字 |
| `*斜体*` | 斜体 |
| `` `コード` `` | インラインコード |
| ` ```コードブロック``` ` | コードブロック |
| `- 箇条書き` | 箇条書き（`*` も可） |
| `1. 番号付きリスト` | 番号付きリスト |
| 行頭に2スペース + `- ネスト` | インデントでネストリスト |
| `- [ ] タスク` / `- [x] 完了` | タスクリスト（チェックボックス） |
| `~~打消し線~~` | 打消し線テキスト |
| `---` | 水平線 |
| `\| col1 \| col2 \|` | テーブル |
| `> 引用` | ブロッククォート |
| `[text](https://...)` | リンク |

### 出欠をカウントする

授業が登録されているセルにマウスを乗せると、「**出席 +1**」「**欠席 +1**」ボタンが表示されます。

- クリックするとモーダルを開かずに即座にカウントが増えます
- 細かく数値を修正したい場合は授業モーダルの「出欠カウント」欄から直接入力・増減できます
- 時間割セルには `出x 欠y` が表示されます

## コマの時間を変更する

1. ヘッダー右端の **⚙ ボタン**をクリック
1. **「コマの時間」タブ**で各限の開始・終了時刻を変更
1. 「保存」を押す
1. 時間割の左列と昼休み行が即座に更新されます

---

## ノートの保存場所を変更する

デフォルトでは `~/Library/Application Support/university-timetable/notes/` に保存されます。

1. ヘッダー右端の **⚙ ボタン**をクリック
1. **「その他の設定」タブ**を開く
1. 「変更」ボタンを押してフォルダ選択ダイアログで保存先を指定
1. 「保存」を押す

- 「Finderで開く」で現在の保存フォルダをすぐに確認できます
- 「リセット」でデフォルトの場所に戻せます
- 変更後は**新規保存から**新しい場所に反映されます（既存ファイルは移動されません）

---

## 大学ポータルへのリンクを設定する

1. ヘッダー右端の **⚙ ボタン**をクリック
1. **「その他の設定」タブ**を開く
1. 「大学ポータル URL」にポータルサイトのURLを入力（例：l-cam のURL）
1. 「保存」を押す

URLを設定するとヘッダーに「**ポータル**」ボタンが表示され、クリックするとデフォルトブラウザで開きます。

## 学期の管理

ヘッダーの「学期管理」ボタンから以下の操作ができます。

### 学期を切り替える

- 学期名をクリックするとその学期の時間割に切り替わります
- ヘッダーのドロップダウンからも切り替えできます

### 新しい学期を作成する

1. 「学期管理」を開く
1. 入力欄に学期名を入力（例：`2026年後期`）
1. 「作成」を押す

### 学期を削除する

- 現在選択中の学期は削除できません
- 削除するとその学期の**授業データもすべて削除**されます

---

## データの保存場所

データはローカルに自動保存されます。アプリを閉じてもデータは保持されます。

| データ | 場所 |
| --- | --- |
| 時間割DB | `~/Library/Application Support/university-timetable/timetable.db` |
| ノート（.md） | `~/Library/Application Support/university-timetable/notes/`（デフォルト） |
| アプリ設定 | `~/Library/Application Support/university-timetable/settings.json` |

回数ごとのノートは `notes/semester_{id}/{day}/p{period}_c{id}_{slug}/session_{n}.md` に保存されます。  
ノートの保存先は⚙ → 「その他の設定」タブで変更できます。

## トラブルシューティング

### アプリが起動しない

```bash
# 依存パッケージを再インストール
npm install
npm run dev
```

### データをリセットしたい

```bash
rm ~/Library/Application\ Support/university-timetable/timetable.db
```

次回起動時に初期設定画面から始まります。
